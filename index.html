Store Master

List every store to be monitored. Output: Store_Master.csv

What to do on this tab:
  1. If you have an existing store list (Excel or CSV), upload it using the box below — the tool will auto-match your columns.
  2. No file? Click "+ Add Row" to add stores one by one.
  3. After uploading, a Column Mapper appears — check each match and click "Apply & Import Rows".
  4. Fix any rows highlighted in red (missing required fields) before downloading.
  5. Optional: name the Custom Columns (e.g. "Zone", "City") if you need extra metadata beyond the standard fields.

SKU Assortment

Upload the product list (SKUs) the app should detect in stores. Output: SKU_Assortment.csv

What to do on this tab:
  1. Choose Assortment Mode below. If unsure, use Category Level — it's the most common setup.
  2. Upload one or more SKU files. Each uploaded file = one assortment. After uploading, give the assortment a name and map the columns.
  3. No file? Click "+ Add Row" to enter SKUs manually.
  4. The flag columns (Core, NPD, Regular OSA, SOS) are optional — enter 1 for yes, 0 for no, or leave blank.
📋 Column reference — SKU Assortment ▼ hide

Category NameRequiredProduct category this SKU belongs to. E.g. Noodles

Store-Assortment Mapping

Link each store to the right product assortment. Output: Store_Assortment_Mapping.csv

⚠️Complete Store Master and SKU Assortment first. Stores and assortment names are pulled from those tabs.
What to do on this tab:
  1. The Smart Assortment Assigner below shows all your assortments (from SKU Assortment tab). Click any assortment block to expand it.
  2. Check the checkbox next to each store that should use that assortment. Fill in Category Name and Shelf Section.
  3. Rows appear automatically in the table below as you check stores — no need to type them manually.
  4. If you added stores or assortments after opening this tab, click "Refresh" to reload the list.
🔗 Smart Assortment Assigner
Assortment names are pulled from SKU Assortment (Tab 4). Stores are pulled from Store Master (Tab 2). Expand each assortment block and check the stores that should use it.
